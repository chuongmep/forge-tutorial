## Before run

Setting file `.env` : 

```
FORGE_CLIENT_ID=<your client id>
FORGE_CLIENT_SECRET=<your client secret>
FORGE_CALLBACK_URL="http://localhost:8080/api/auth/callback"
SERVER_SESSION_SECRET="custom-encryption-key"
```


## How to run 

command : 

```
npm install
npm run start 
```

Or run by command : 

```bash
node server.js
```
